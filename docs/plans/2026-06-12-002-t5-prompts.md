## T5.1 — 修复 Demo 脚本

```
### 任务：修复 Demo 脚本使其能在当前引擎上运行

### 当前状态
- scripts/demo.lua + scripts/demo_story.ks 存在
- 未在 CI 中验证过
- 引擎 API 经过大规模重构后 Demo 可能已不兼容

### 执行

#### Step 1: 尝试运行 Demo
- 在 headless 模式下尝试加载 Demo 脚本
- 通过 LuaManager 加载 demo.lua → 执行
- 记录所有错误（API 名称变更、参数不匹配等）

#### Step 2: 修复 API 调用
- 根据当前 src/script/bindings/ 中的实际 API 修复 Demo
- 确保所有 KAG 命令使用当前支持的语法
- 确保所有 Lua API 调用使用正确的模块名

#### Step 3: 简化 Demo
- 只保留核心 galgame 流程：
  1. 显示标题画面
  2. 一段对话（@text）
  3. 背景切换（@bg）
  4. 选项分支（@choice）
  5. 结局画面
- 移除依赖外部资源的调用（图片/BGM 用占位符）
- 提交

### 预期产出
- scripts/demo.lua + scripts/demo_story.ks 可在当前引擎上运行
- 2 次提交
```
