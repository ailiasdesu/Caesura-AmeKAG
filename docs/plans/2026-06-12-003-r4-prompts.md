```
### 当前状态
- script 模块 9 cpp + 9 h，1 个 api 接口（ILuaManager）
- 有 test_lua_manager.cpp, test_kag_binding.cpp, test_kag_integration.cpp, test_kag_execution.cpp
- 缺失：DevCore/Render/VFX/Steam bindings 的独立测试
- 缺失：GameState 的独立测试

### Step 1: 绑定测试
- 创建 tests/cpp/test_script_bindings.cpp
- 测试用例（至少 6 个）：
  1. DevCoreBinding get_version 返回字符串
  2. DevCoreBinding get_backend_name 返回 "bgfx"/"SoLoud" 等
  3. RenderBinding load_texture 无效路径返回 0
  4. VFXBinding create_emitter 返回有效 id（>0）
  5. VFXBinding alive_count 初始为 0
  6. KAGBinding play_bgm 无效文件返回 0
- 每个测试前需要初始化 BackendRegistry（注册 Null 后端）
- 更新 tests/CMakeLists.txt
- 构建 + 测试全绿后提交

### Step 2: GameState 测试
- 创建 tests/cpp/test_game_state.cpp
- 从 Lua 端测试 GameState（通过 LuaManager 执行 Lua 脚本）：
  1. GameState push "key" → 同一协程 get 返回相同值
  2. 未 push 的 key 返回 nil
  3. GameState clear 后所有 key 返回 nil
  4. 两个协程之间的 GameState 隔离
- 更新 tests/CMakeLists.txt
- 构建 + 测试全绿后提交

### 预期产出
- tests/cpp/test_script_bindings.cpp（6+ 用例）
- tests/cpp/test_game_state.cpp（4+ 用例）
- 2 次提交
```
