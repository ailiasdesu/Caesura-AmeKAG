## Step 1: 实现 RTT 读回

```
文件：src/storage/SaveManager.cpp

修改 captureThumbnailPNG()：
- 通过 BackendRegistry::instance().getRenderDevice() 获取渲染设备
- 创建临时 RTT（320×180）
- 使用 bgfx::setViewRect + bgfx::touch 设置视图
- 调用 bgfx::readTexture 异步读回 RGBA 数据
- 等待一帧（bgfx::frame()）确保数据就绪

关键约束：
- headless 模式下渲染设备可能不可用 → 返回空字符串（安全降级）
- readTexture 是异步的，必须等待 frame() 后才能访问数据
- RTT 使用完毕后销毁

构建 + 测试全绿后提交。
```
