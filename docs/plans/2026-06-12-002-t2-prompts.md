```
### 任务：提取平台初始化阶段

### 当前状态
- Engine::init() 前 ~100 行创建 SDL3 平台 + bgfx 渲染设备
- 代码大致流程：SDL3 init → bgfx init → BackendRegistry 注册 → FreeType init

### 执行

#### Step 1: 提取平台初始化
- 读取 src/entry/Engine.cpp 中 init() 的平台初始化部分
- 创建私有方法：
  ```cpp
  bool initPlatformPhase(int width, int height, bool headless);
  ```
- 包含：
  - SDL3PlatformBackend 创建和 init
  - BgfxRenderDevice 创建和 init
  - 注册到 BackendRegistry（setPlatformBackend/setRenderDevice）
  - FreeTypeContext init
  - TextureBudget detect
- 失败路径：清除已创建的 unique_ptr
- Engine::init() 调用 initPlatformPhase()，失败则返回 false
- 构建 + 测试全绿后提交

#### Step 2: 添加平台阶段测试
- 扩展 tests/cpp/test_engine_lifecycle.cpp
- 测试用例：
  1. EngineConfig 设置 headless=true → init → shutdown（完整 headless 生命周期）
  2. 平台初始化失败时 Engine 正确返回 false
- 提交

### 预期产出
- Engine.cpp 中新增 initPlatformPhase() 私有方法
- 2 次提交
```
