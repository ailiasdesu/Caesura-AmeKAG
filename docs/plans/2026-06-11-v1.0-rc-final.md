# v1.0-rc 最终打磨

## Summary

引擎 ~95%，CI 三平台全通。v1.0-rc 收尾：存档完善、移动端接口预留、编辑器体验打磨。

## Implementation Units

### IU-1: SaveManager 缩略图优化
- 当前：`captureThumbnailPNG` 用 `bgfx::requestScreenShot` 写磁盘再读回（功能可用）
- 目标：bgfx 是唯一截图 API，磁盘方式对存档场景（低频操作）可接受
- 优化方向：添加内存缓存避免重复 I/O，resize 到 320×180
- 文件：`src/System/SaveManager.cpp`

### IU-2: MobileAdapter 接口完善
- 当前：P2 存根，触屏事件预埋
- 目标：补充完整的触屏→引擎事件映射、虚拟键盘、安全区域适配
- 不需要实际移动端运行，确保接口完备即可
- 文件：`src/Core/MobileAdapter.cpp`

### IU-3: 编辑器体验打磨
- StageView 帧预览流畅度（降低轮询间隔或改用事件驱动）
- LogPanel 搜索 + 导出
- 快捷键完善（F5 运行、F11 全屏等）
- 文件：`web-editor/src/`

### IU-4: 文档 & Demo
- Demo 场景增加存档/读档交互展示
- README 添加 getting started 教程链接
- KAG-API.md 添加 Lua 命令→C++ API 对照索引

## Test Plan
