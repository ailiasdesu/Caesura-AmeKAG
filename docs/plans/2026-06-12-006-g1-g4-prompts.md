## G1: 制作真正的 Demo Galgame

```
这是当前最重要的事——验证引擎在真实场景中可用。

### G1.1: 资源准备

用 imagegen 或公开素材生成：
- 3 张背景：教室(1920×1080)、走廊(1920×1080)、屋顶(1920×1080)
- 2 张立绘：角色A（正常表情）、角色A（微笑）
- 1 首 BGM：30 秒循环 WAV
- 2 个音效：翻页声、点击声
放 assets/demo/ 目录。

### G1.2: Demo 脚本

扩展 scripts/demo_story.ks，使用真实资源路径：

*start
播放 BGM
教室场景：主角独白 + 角色A 进入
选项：打招呼 / 无视
分支A：走廊 → 对话 → 结局A
分支B：教室 → 对话 → 结局B
@return 回到标题
屋顶场景（隐藏路线）：特定条件触发 → 结局C

目标：5 分钟阅读流程，3 个结局。

### G1.3: Demo 运行测试

tests/cpp/test_demo_e2e.cpp 扩展：
- Demo 从头跑到尾不崩溃
- 至少 3 个 @choice 被触发
- 3 个结局均可达
```
