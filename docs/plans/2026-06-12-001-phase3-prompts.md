  BgfxRenderDevice(3), ParticleSystem(3), FreeTypeContext(3),

  tests/cpp/test_render_device.cpp — BgfxRenderDevice + RTTManager + EmbeddedShaders
  tests/cpp/test_render_pipeline.cpp — TextRenderer + FreeTypeContext + ShaderCache
  tests/cpp/test_particle_system.cpp — ParticleSystem（已存在，合并内容）
  tests/cpp/test_layer_manager.cpp — LayerManager + GpuMonitor
  tests/cpp/test_texture_manager.cpp — TextureManager + VideoPlayer
- 检查 tests/cpp/test_particle_system.cpp 是否已有内容，避免重复
- 从 test_render.cpp 迁移 TEST_CASE 到对应新文件
- 删除原 test_render.cpp（所有测试已迁移）
- 更新 tests/CMakeLists.txt
- 构建 + 测试全绿后提交

#### Step 2: BgfxDraw.cpp 进一步拆分（333 → 目标 <200 行）
- 分析 BgfxDraw.cpp 中的 11 个函数：
  init, beginBatch, flushBatch, blitTexture(x2), stretchBlt, affineBlt,
  submitBlend, submitTransition, submitVFX, fillViewport
- 提取方案：
  - BgfxDraw_Batch.cpp: init, beginBatch, flushBatch
  - BgfxDraw_Blit.cpp: blitTexture(x2), stretchBlt, affineBlt

- 使用 DrawState* 作为共享上下文（已经存在）
- BgfxDraw.cpp 保留为转发文件或删除（所有函数已迁移）
- 更新 CMakeLists.txt
- 构建后确认 BgfxRenderDevice 仍能正常编译和委托
- 提交

#### Step 3: 接口中去 bgfx 化（第一轮）
- 当前接口使用了 bgfx 具体类型：
  IRenderDevice: bgfx::ProgramHandle, bgfx::UniformHandle, bgfx::VertexLayout
  ITextureManager: bgfx::TextureHandle
  ILayerManager: bgfx::TextureHandle, bgfx::ProgramHandle
  IVideoPlayer: bgfx::TextureHandle
  IParticleSystem: 无 bgfx 类型暴露 ✅
- 方案：创建不透明句柄包装
  - src/render/api/RenderTypes.h 定义：
    struct TextureHandle { uint32_t id = 0; };
    struct ProgramHandle { uint32_t id = 0; };
    struct UniformHandle { uint32_t id = 0; };
  - 接口方法改用这些不透明句柄
  - BgfxRenderDevice 内部维护 bgfx handle → 不透明 handle 的映射
- **这一步是可选的高风险重构，可以标记为 "Deferred"。**
  如果执行，需要更新所有消费者（RenderBinding, KAGBinding, VFXBinding 等）。
  建议：先创建 RenderTypes.h 和不透明句柄，逐步迁移接口方法。
- 提交（至少完成 RenderTypes.h 定义）

#### Step 4: 添加功能性测试
- 在拆分的测试文件中添加有实际验证逻辑的测试：

  test_texture_manager.cpp:

  - createSolidTexture 返回非 BGFX_INVALID_HANDLE
  - destroyTexture 无效 id 不崩溃
  
  test_particle_system.cpp:
  - createEmitter 返回有效 id
  - destroyEmitter 无效 id 不崩溃（粒子系统已存在时）
  - update 后 aliveCount ≥ 0
  
  test_layer_manager.cpp:
  - setVisible/setOpacity/setPosition 不崩溃
  - markDirty 后 dirty rect 非空
- 构建 + 测试全绿后提交
- 期望测试数：149 → 165+
