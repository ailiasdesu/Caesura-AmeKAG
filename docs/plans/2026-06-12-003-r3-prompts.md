## R3.2 — CARC 打包指南

```
### Step 1: 创建打包指南
- 创建 docs/guides/carc-packaging.md
- 从 tools/carc_pack/main.cpp 提取 CLI 用法
- 从 src/archive/CARCWriter.cpp 提取打包流程

内容模板：
# CARC 打包指南

## 概述
CARC (Caesura ARChive) 是引擎的加密压缩归档格式。支持 AES-256-GCM 加密 + zstd 压缩 + Ed25519 签名。

## 打包命令
tools/carc_pack <输入目录> <输出.carc> [--key <密钥文件>]

## 示例
打包包含加密：
tools/carc_pack ./mygame ./release/game.carc --key ./mygame/game.key

不加密：
tools/carc_pack ./mygame ./release/game.carc

## 密钥生成
tools/carc_pack --genkey ./mygame/game.key
生成 game.key（AES-256 密钥）+ game.key.pub（Ed25519 公钥）

## 在 KAG 脚本中引用
@bg "carc://scenes/bg_classroom.png"
@bgm "carc://audio/bgm_01.mp3"

## 格式结构
Header (128B): 魔数 + 版本 + 压缩算法
Index  (加密): 文件路径 → 偏移量 + 大小
Body   (加密): 文件数据
Footer (128B): Ed25519 签名 + 公钥

- 提交

### 预期产出
- docs/guides/carc-packaging.md
- 1 次提交
```
