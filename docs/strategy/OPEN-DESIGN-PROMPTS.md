# Open Design 提示词 — Caesura 编辑器 GUI

> 用法：在 Open Design 中打开 Caesura 项目（含 DESIGN.md），逐个粘贴下方提示词。
> 生成的原型保存为 HTML，我来翻译到 React 组件。

---

## 提示词 1：编辑器整体框架

```
为 Caesura 视觉小说引擎生成编辑器整体框架布局页面。

要求：
1. 严格遵循项目 DESIGN.md 的配色和间距。
2. 顶部 36px 工具栏：左侧引擎名+状态指示灯（绿=已连接，红=断开），中间面板切换标签（Stage/Code/AI），右侧窗口控制按钮。
3. 左侧 200px 素材浏览器：文件夹树（bg/char/bgm/voice/se/scripts），支持拖拽素材到舞台。
4. 中央舞台区域（flex:1）：深色画布，1280×720 安全区虚线框，网格背景。舞台底部浮动工具栏（Grid切换/Zoom缩放/分辨率切换/Preview按钮）。
5. 右侧 240px 属性面板：选中素材时显示位置(x,y)、缩放、透明度、图层(bg/fg/msg)编辑。
6. 底部 120px 时间线：水平排列的事件块（图/文/音/特效），不同颜色区分事件类型，支持拖拽排序。
7. 底部 160px 可折叠日志/AI面板（tab切换）。

只输出单个完整 HTML 文件。
```

## 提示词 2：Stage 舞台视图

```
为 Caesura 编辑器生成 Stage 舞台视图组件。

要求：
1. 1280×720 画布居中显示，深色背景 #1a1a24。
2. 40px 网格线 #1e1e2a。
3. 128px 安全区虚线框 #2a2a3a。
4. 底部浮动工具栏：Grid 开关按钮、Zoom +/- 按钮、分辨率切换按钮(1280×720/1920×1080)、Preview 按钮（三角形图标）。
5. 拖入素材时显示蓝色虚线边框 + glow 效果。
6. Preview 激活时按钮变绿色实心。

输出单个 HTML 文件。
```

## 提示词 3：AI 面板

```
为 Caesura 编辑器生成 AI 辅助面板组件。

要求：
1. 顶部：AI 后端选择器（OpenAI/Codex/Custom 三选一下拉）。
2. 消息列表：用户消息右对齐（#2a2a4a 气泡），AI 回复左对齐（#1c1c28 气泡），流式输出时最后一条消息带闪烁光标。
3. 底部输入区：textarea + 发送按钮（紫色强调色）。
4. 快捷指令按钮行：@generate（生成 KAG 代码）、@fix（修复错误）、@explain（解释代码）。
5. 生成中状态：骨架屏 shimmer 动画。
6. 代码块用深色背景 #0a0a0f + 语法高亮色。
7. 设置齿轮图标→弹出 SettingsDialog。

输出单个 HTML 文件。
```

## 提示词 4：时间线

```
为 Caesura 编辑器生成时间线组件。

要求：
1. 水平滚动的时间轴，120px 高。
2. 事件块用不同颜色区分：紫色=show_image，蓝色=show_text，绿色=play_bgm，橙色=transition，青色=video。
3. 每个事件块显示：图标 + 标签（截断到 20 字）+ 时长条。
4. 支持拖拽排序（拖拽时半透明 + 插入位置指示线）。
5. 选中事件块时高亮边框 + 属性面板联动显示详情。
6. 搜索/过滤输入框在左上角。
7. 累计时间刻度尺在顶部（每 5 秒标记）。

输出单个 HTML 文件。
```

## 提示词 5：代码编辑器

```
为 Caesura 编辑器生成 Lua 代码编辑器组件。

要求：
1. 深色背景 #0a0a0f，JetBrains Mono 13px。
2. 行号左侧栏（#1c1c28 背景，#555570 文字）。
3. 语法高亮：关键字紫色、字符串绿色、注释灰色、数字橙色、函数名蓝色。
4. 错误行左侧红色竖线标记 + 底部浮动错误提示。
5. 与属性面板双向同步：编辑属性→代码自动更新，编辑代码→属性自动刷新。
6. Ctrl+S 保存提示（底部短暂 toast）。
7. 顶部文件标签栏（main.lua / scene1.lua / ...）。

输出单个 HTML 文件。
```
